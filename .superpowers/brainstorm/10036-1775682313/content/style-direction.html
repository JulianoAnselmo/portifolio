A — Minimal Dark

Ultra-limpo, sem distrações. Foco total nos projetos com bastante espaço em branco (escuro). Tipografia forte, poucos elementos decorativos. Referência: portfólios de designers de produto.

Juliano Anselmo
Desenvolvedor Web

B — Glass & Gradients

Cards com efeito vidro (glassmorphism), gradientes sutis no fundo, blur e transparências. Visual sofisticado e moderno. Referência: Apple, Linear, Vercel.

Juliano Anselmo
Desenvolvedor Web

C — Bento Grid

Grid assimétrico estilo "bento box" com cards de tamanhos variados. Projetos em destaque ficam maiores. Layout dinâmico e visualmente interessante. Referência: GitHub profile, Apple.
